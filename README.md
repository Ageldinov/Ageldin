# Ageldin

**Project Name: Ageldin**

**Author: Ageeldinov Ruslan**

**Date of creation: 06/01/2023**

**General description: **

This set of scripts is designed to perform simple calculations.
At the moment, it includes: a convector of dimensions (from pkg to mg), a basic one-component buffer calculator, a simplified version of the calculation of pharmacokinetic parameters and visualization of concentration changes over time, a tool for finding coordinates on a given image (in two dimensional space) and basic calculations for primers (in development).

The project was created to simplify calculations and strives to collect all possible tools in one shell.

**Related files:**

The project contains one main file: Ageldin.py. This is a basic file that unites all the created modules and contains the structure and description of the visual component with prescribed sequences of actions.
The package also includes additional modules: PK_calculation.py, buffer_calculation.py, convertation.py and points_coordinates.py. Each of them is responsible for the sequence of actions in the corresponding tab.

**Общее описание: **

Данный набор скриптов предназначен для осуществления простых вычислений.
На данный момент он включает в себя: конвектор размерностей (от пкг до мг), базовый однокомпонентный буферный калькулятор, упрощенный вариант расчетов фармакокинетических параметров и визуализации изменения концентрации во времени, инструмент поиска координат на заданном изображении (в двух размерном пространстве) и базовые расчеты для праймеров (в разработке).

Проект создан для упрощения вычислений и стремится к сбору всех возможных инструментов в одной оболочке.

**Связанные файлы:**

Проект содержит один основной файл: Ageldin.py. Это базовый файл объединяюший все созданные модули и содержащий в себе структуру и описание визуальной составляюшей с прописанными последовательностями действий.
В состав также входят дополнительные модули: PK_calculation.py, buffer_calculation.py, convertation.py и points_coordinates.py. Каждый из них отвечает за последовательность действий в соответствуюшей вкладке.
